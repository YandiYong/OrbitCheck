## Getting Started

1. Simply open `index.html` in your web browser
2. Start adding tasks using the input field
3. Check off tasks as you complete them
4. Delete tasks you no longer need

## Usage

### Adding a Task
- Type your task in the input field
- Click "Add Task" or press Enter

### Completing a Task
- Click the checkbox next to a task to mark it as complete
- Click again to mark it as incomplete

### Deleting Tasks
- Click the "Delete" button on individual tasks
- Use "Clear Completed" to remove all completed tasks
- Use "Clear All" to remove all tasks (requires confirmation)
